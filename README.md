
# Simple Password Generator (GUI Version)

This project is a Python-based Password Generator using multiple classes and a GUI.

## Project Structure
- password_generator.py : Contains PasswordGenerator class
- gui.py : Contains PasswordGUI class
- README.md : Project documentation

## Features
- GUI using Tkinter
- Random strong password generation
- Multiple class structure

## How to Run
1. Install Python
2. Run the GUI:
   python gui.py
